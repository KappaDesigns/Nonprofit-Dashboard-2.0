Module: util/Logger

The logger module is built on top of the Winston logging module Winston. This module allows for loggers with different tags and debuging levels to be created with ease.

(require("util/Logger"))(tag, debugLevels, testingEnv) → {Logger:Object}

Creates a new Logger using a given tag, debugging levels, and environment. Allowing multiple loggers to be easily created through out the application.
Parameters:
Name Type Description
tag String String that defines the tag of the logger and allows the logger to be easily identified
debugLevels Array.<String> array of strings that determines what level of logs should be written as files to the logs directory
testingEnv Boolean determines if the current logger is running in a testing env or not
